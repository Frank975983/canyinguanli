销售数据录入

本页供门店员工日常录入销量。请先在右上角选择具体分店，再选择产品与渠道。

今日已录记录

仅显示当前分店今日录入，不含售价与营收信息。员工不可删除数据。

物料采购申请

仅填写原料库品类与需求数量，不含金额。提交后由超级管理员核销入账。


待核销申请

员工已提交、采购完成后在此核销：填写实付金额入账；若遇烂货退货可对已核销申请「撤销核销」。

我的申请记录

可查看申请状态；已核销后显示核销时间，不展示金额。员工不可撤销或删除。

费用录入

填写费用后提交，数据将自动进入超级管理员「财务对账」费用明细。请先选择具体分店。

已提交费用

当前分店已录入的费用记录，管理员可在财务对账中查看并核算。员工不可删除。

使用说明

1. 先在「门店管理」确认门店 → 2. 「原料库」录入原料 → 3. 「产品信息」配置 BOM → 4. 「产品销售」录入或导入 Excel → 5. 查看「物料统计」和「财务对账」
无需安装、无需服务器，数据存在本机浏览器中。

添加原料

分店视图下添加本店专属原料；总店视图可添加「共用原料」（如餐盒、餐具）并分配到各分店。

原料类型

共用原料只需填写名称。单位、规格、价格、入库与分配请到下方「共用原料（总店）」按先入先出处理。

共用原料（总店）

在此维护规格与价格，并办理入库、分配。库存按先入先出（FIFO）扣减：优先消耗最早入库批次；可分配库存为 0 时不可再分配。

FIFO 入库批次
